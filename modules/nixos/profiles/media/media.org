#+TITLE: Media Profile
#+AUTHOR: Inkpotmonkey

* Overview
The Media profile orchestrates a complete media stack including consumption (=Jellyfin=), automation (=FlexGet=), and acquisition (=Transmission=).

* Components
** Jellyfin
The primary media server. It uses the =jellyfin= group to share access with automation tools.

** FlexGet
The automation engine that monitors feeds and sends download requests to Transmission.
- *Configuration*: Located at =/var/lib/flexget/config.yml= (managed by Nix).
- *Scheduling*: Integrated with systemd timers for periodic execution.

** Transmission
The secure download client. See [[file:transmission.org][transmission.org]] for details on VPN and security.

* Automation Workflows
** Subtitle Generation
The profile includes a custom =auto-sub= script (wrapping Whisper/Transcriber) that automatically generates subtitles for downloaded content.

** Permissions
All services share the =jellyfin= group to ensure seamless file access:
- *Jellyfin*: Reads media.
- *FlexGet*: Moves files and triggers subbing.
- *Transmission*: Writes downloads.

* State Preservation
This profile is designed for *Impermanence*. State is persisted at:
- =/var/lib/jellyfin=
- =/var/cache/jellyfin=
- =/var/lib/flexget=
- The configured =mediaPath= (default: =/var/lib/media=)

* Gotchas & Troubleshooting
** Impermanence Permissions
When using =impermanence=, state directories (especially for Jellyfin and FlexGet) must be initialized with the correct ownership (typically =jellyfin:jellyfin= or =flexget=) to prevent startup failures after a system wipe.

** Automation Latency
Subtitle generation is an asynchronous process managed via =taskspooler=. It may take several minutes after a download completes before the subtitles are available in the media library.

* Fast Configuration Validation
To quickly validate the FlexGet configuration schema without running a full NixOS integration test, use the following one-liner:
#+BEGIN_SRC bash
nix-shell -p flexget -p coreutils --run "flexget -c \$(nix build .#nixosConfigurations.kelpy.config.services.flexget.configFile --no-link --print-out-paths) check"
#+END_SRC
